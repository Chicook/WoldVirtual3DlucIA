# 🔧 Backend - Metaverso Crypto World Virtual 3D

## Descripción

API backend del metaverso con soporte para blockchain, tiempo real y IA.

## Características

- 🔌 API RESTful
- ⚡ WebSocket en tiempo real
- ₿ Integración blockchain
- 🤖 IA LucIA
- 🔐 Autenticación JWT

## Desarrollo

```bash
cd backend
npm install
npm run dev
```

## Tecnologías

- Node.js
- Express
- MongoDB
- Redis
- Socket.io
